# 找不到模块“axios”或其相应的类型声明

*2026/3/11*  
file：`frontend/src/App.vue`  
`import axios from 'axios'//从 axios 库导入 axios`  

原因：  
只有后端backend路径安装了axios，需要在前端frontend路径也安装。 

解决：  
进入frontend，安装axios：  
```bash
npm install axios
```

# 将github token字段添加入commit中后，修改文件也无法进行后续提交

token已经进入历史commit。  
最简单的方法：直接进入github提供的强制允许push password链接：

```
To push, remove secret from commit(s) or follow this URL to allow the secret. remote: https://github.com/...540
```

更改设置后提交，然后禁用提交过的token。

# 修改isUpItem判断逻辑后报错

*2026/3/31*  
decription：修改了isUpItem.js的判定逻辑后报错：Bind parameters must not contain undefined. To pass SQL NULL specify JS null  

```js
function isUpItem(record) {
    if (record.qualityLevel === 5) {
        if (PERMANENT_CHARACTERS.includes(Number(record.resourceId))) {
            console.log('record.resourceId:', record.resourceId);
            return false;
        }
        return true;
    }
}
```

reason：只考虑了qualityLevel===5的数据项。对于该属性值为3或4的数据项，须返回null。
