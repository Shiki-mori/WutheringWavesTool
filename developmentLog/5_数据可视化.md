# 数据可视化

最激动人心的时刻要来力。

设定技术栈使用：`vue3+TypeScript+ECharts+vue-echarts`

## 基础知识

数据映射：字段映射到图表  
图表三要素：标题/坐标轴/数据系列  
ECharts 核心配置：title，tooltip，legend，xAxis，yAxis，series  
响应式更新：接口数据变化后，重新生成option  
组件化：把每张图单独拆成一个Vue组件

## ECharts配置

进入frontend目录，执行：

```bash
npm install echarts vue-echarts
```

## 大纲

先定义页面需要展示什么信息。

暂定先做一个抽卡分析总览页。将后端返回的分析结果完全表示出来。  
统计卡片区：展示总抽数，五星数，平均数等最核心的数字。    
单卡池五星统计图：横轴抽数，纵轴卡池。  
限定池up率环形图  

工程学习路线：  

1. 先在页面里看懂后端返回的数据
2. 做纯文字统计卡片
3. 做最简单的柱状图
4. 做其他类型统计图
5. 页面整理
